POLÍTICA DE PRIVACIDAD DE GASFINDER

Última actualización: 11 de diciembre de 2024

GasFinder ("la Aplicación") es desarrollada por GONZALO RAEZ ZAFRA como aplicación gratuita.

1. INFORMACIÓN QUE RECOPILAMOS

La Aplicación accede a:

  • Ubicación del dispositivo: Para mostrar gasolineras cercanas y calcular distancias
  • Datos de gasolineras: Obtenidos de APIs públicas del Gobierno de España

2. USO DE LA INFORMACIÓN

  • La ubicación se utiliza únicamente para calcular distancias a gasolineras
  • NO almacenamos tu ubicación en servidores externos
  • NO compartimos tu información con terceros
  • Los datos se procesan localmente en tu dispositivo

3. ALMACENAMIENTO DE DATOS

  • La app guarda preferencias localmente (gasolineras favoritas, filtros)
  • Caché temporal de precios de combustible
  • Ningún dato personal se envía a servidores externos

4. PERMISOS

  • Ubicación: Necesario para la funcionalidad principal
  • Internet: Para obtener precios actualizados de combustible

5. SERVICIOS DE TERCEROS

La aplicación utiliza:

  • API pública de precios de combustible del Ministerio de Industria de España
  • Google Maps para visualización de mapas (sujeto a su propia política de privacidad)

6. SEGURIDAD

Tomamos medidas razonables para proteger la información procesada por la aplicación.

7. CAMBIOS EN ESTA POLÍTICA

Cualquier cambio en esta política se notificará mediante actualización de la aplicación.
